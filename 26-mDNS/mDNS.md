## [mDNS](#)

  Это пример HTTP-сервера, доступ к которому осуществляется по URL-адресу http://esp32.local благодаря mDNS-серверу. 

  Инструкции:
  
  - При необходимости измените SSID и пароль Wi-Fi. 
  
  - Загрузите скетч на плату ESP32.
 
  - Установите программное обеспечение для хоста: 
 
    - Для Linux установите Avahi (http://avahi.org/).
    - Для Windows установите Bonjour (http://www.apple.com/support/bonjour/).
    - Для Mac OSX и iOS поддержка уже встроена в Bonjour.
   
  - Откройте в браузере http://esp32.local, вы должны увидеть ответ.

> ***В Андроиде ни Bonjour, ни Avahi нет. Нужно устанавливать другие приложения для того, чтобы срабатывало mDNS для Esp32.***
